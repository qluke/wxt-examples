# 基于 AI 解释 WXT 示例核心功能

示例位于 [`examples/` 目录](./examples)。

要下载并在本地运行特定示例：

```sh
npx giget@latest gh:wxt-dev/examples examples/<示例目录名>
cd <示例目录名>
```

然后按照示例 README 文件中的说明运行它。
